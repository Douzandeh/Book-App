# 📚 Book App

A modern book discovery application built with React. Users can browse books, search for their favorite titles, and explore book details through a fast and responsive interface.

## 🚀 Features

* Browse books
* Search functionality
* Responsive design
* Dynamic UI with React
* Clean and user-friendly interface

## 🛠️ Tech Stack

* React
* JavaScript (ES6+)
* CSS3
* Vite

## 📦 Installation

```bash
git clone https://github.com/Douzandeh/Book-App.git
cd Book-App
npm install
npm run dev
```
